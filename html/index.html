Welcome to the Docker Webinar!

This is a page is served to you by NGINX running in Docker container.

docker run --name some-nginx -v /Users/alexlanda/work/docker/webinar/html:/usr/share/nginx/html:ro -p 80:80 nginx

some-nginx is the container name

the container is listening on port 80

the container mounts /Users/alex.../html as /usr/share/nginx/html
